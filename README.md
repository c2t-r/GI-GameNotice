## Usage
1. Fork this repo
2. Add secrets
```ini
NAME = GI-GameNotice
WEBHOOK = https://discord.com/api/webhooks//
LANGUAGE = ja // en, ja, es....
```
![image](https://github.com/c2t-r/GI-GameNotice/assets/80561604/63d8a4f2-9ec2-49d7-a637-44d728b2f945)
