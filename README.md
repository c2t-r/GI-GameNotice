# GI-GameNotice

## Usage
1. Fork this repo
2. Add secrets
```ini
NAME = GI-GameNotice
WEBHOOK = https://discord.com/api/webhooks//
LANGUAGE = ja // en, ja, es....
```
3. Set `Workflow permissions` to `Read and write permissions`
4. Enjoy!
